# My Portfolio Site  
ポートフォリオサイトです。私の自己紹介や技術スタック、活動内容を記載しています。   
Next.jsのチュートリアルで学んだことのアウトプットとして制作し、Vercelでデプロイしました。   
参照：[サイトに飛びます](https://kyoheiyonehama.vercel.app/)
## 技術スタック
- Next.js    
- TypeScript   
- tailwindCSS  
- tsParticles   
- ResendAPI   
- AtCoderが提供しているエンドポイント   
## 今後の展望   
- DBを用いてブログ部分も個人ページにする   
  - マークダウンで編集したものをHTMLでレンダリングする
  - 投稿ページにログイン認証を用いる
  - 画像の保存も可能にする
  - ログインせずに匿名コメントを可能にする
  - supabaseを使用
- 今は各情報を変数で持っているがこれもDBに移管して変更のたびに再デプロイしないで済むようにする
